Proyectos

Proyecto (implementación de Wsus)

Wsus ó Windows Server Update Services, permite a los administradores, gestionar y distribuir las actualizaciones mediante una consola MMC. Este necesita conectarse a Microsoft Update para recuperar las actualizaciones disponibles.

Desde un servidor local se administran aproximadamente 450 equipos que se encuentran actualmente activos dentro de la empresa, con esto se ha evitado que los usuarios no actualicen su equipo sin actualizacion previa.

Gestión de la infraestructura de TI

La gestión de la infraestructura de TI implica principalmente la administración y gestión de componentes de TI críticos para emplear eficazmente la información, los datos y la tecnología. Una infraestructura de TI da forma al entorno de TI de una empresa. Está conformada por todos los componentes que facilitan la gestión y el uso de los datos.

Actualmente dentro de Petstar se apoyo para poder pasar los servicios que se tenian locales como servidores a Gestiónarlos a la nube usando actualmente Amazon Cloud (Active directory).

Creacion de aplicacion para el monitoreo de red en las Plantas

Se creo una aplicacion para monitorear todas las redes en la empresa, ya que al contar con mas de 20 plantas es indispensable el internet, con dicha aplicacion al momento que no responde algun servicio se procede a reportarlo con el proveedor.

implementación de Mesa de ayuda (Service Desk)

Se implemento el uso de una mesa de ayuda para el apoyo a usuarios, esto para que al momento que tuvieran algun problema con su equipo se reportara en esta plataforma y con el uso de metodologias como ITIL para hacer mas eficiente la gestión de servicios de tecnologías de la información
